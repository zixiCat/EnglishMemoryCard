# EnglishMemoryCard

A spaced-repetition flashcard app for studying English vocabulary notes.

## Review Schedule

Clicking **Remember** advances a card to the next stage and schedules it based on the forgetting curve:

| Review # | Stage transition | Next review in |
| -------- | ---------------- | -------------- |
| 1st      | 0 → 1            | 1 day          |
| 2nd      | 1 → 2            | 3 days         |
| 3rd      | 2 → 3            | 7 days         |
| 4th      | 3 → 4            | 14 days        |
| 5th      | 4 → 5            | 30 days        |
| 6th      | 5 → 6 (mastered) | 60 days        |
